This Loan Management app is an incredibly powerful and versatile loan app that help you to manage loan with invoices, Journal Entries, Outstanding balance, principal and interest balalnce, payments etc. Also provides different ways to pay loan installments and provides functionality to change loan installments date.





Hot Features
  • Auto creation of loan installments

    Scheduler auto creates loan installment invoice base on date.

  • Postpone Installments

    Postpone installments by certain amount of days or months, also reset it if need. By certain amount of month postpone installment allow to postpone by selecting loan installment's items.

  • Extra Payment

    Pay extra amount of loan with some options. Like cover that amount only in principal amount or in installments or both.

  • Close Loan

    Don’t allow to close if there is any partial or paid loan iinstallments invoice.Close loan functionality cancel all draf and open loan installments invoices. Add ‘C’ after loan name.

  • Cancel Loan

    Cancel loan funcionality cancel all draft and open loan installments invoices. Marks partial paid loan installments invoices as fully paid with Write-Off account.

  • Report

    PDF report to print loan details

  • Send Loan Detail to customer via Email

    User can send email for loan detail with pdf file.

  • Server Actions

    1. To update loan installments invoices number.
    2. To Create loan installment invoices manually before today date.

Key Features

  • Provides all details like Transaction history, Payment history, Outstanding balance, Invoices, Total due amounts, Due days, Paid tax, insurance, principal, interest in one view.

  • Transaction history of loan installments invoices and its payments with current date balance. Creates transaction history on validation of loan installment invoice and also removes it when cancel loan installment invoice.

  • Payment history of all loan payments and linked invoices with payments. Creates payment history when paying amount and removes when cancling payment. Remove invoice link from payment history on unreconcilation of payment from invoice.

  • Differenciate loan products by addind ‘Propety’ category inside product.

  • Creates Processing fee invoice when validating loan.

  • Displays Loan details and Payment amount details from customer view so user can see total loans and total payment amounts of any customer.

  • Displays Total Cashed and Paid details from product view.

Postpone Installments
- Allow to postpone loan installments date by certain number of days or month by selecting specific date.
- This functionality postpone open loan invoices due date and invoice date or postpone installments date which are after selected date.
- By certain month option user has choice to postpone whole invoice or want to postpone invoice by selecting number of items(like Principal, Interest, Tax or Insurance).
- Also provides reset functionaly that will reset all instllments dates and invoices.
- Adds history for all intallment changes.





Extra Payment
- Allow to pay loan amount in different way like,
1. Settle with next due instalment rest with principal
2. Cover all next due installments
3. Cover only principal outstanding

1. Settle with next due instalment rest with principal
- Using this way user can pay first installment and if amount greater than first installment then rest pays in principal amount.

2. Cover all next due installments
- Cover all next due installments option pays all amount in installments(Loan invoices).

3. Cover only principal outstanding
- To pay only in principal amount.

- This also changes installment schedule based on payment amount and payment method.
- Also provide functionality to close loan, this will calculate total open invoices amount plus remaining principal. So user can pay all remaininging amount and close loan directly.
